# Kitchen Display System (KDS)

## Purpose

Provides chefs with a real-time queue of food preparation.

---

## Features

- Kitchen queue
- Preparation tracking
- Ready notifications
- Order timing
- Chef assignment
- Kitchen analytics

---

## Workflow

Order

↓

Kitchen Ticket

↓

Preparing

↓

Ready

↓

Served

# Promotions Module

## Purpose

Provides promotional tools to increase customer engagement.

Supports:

- Discounts
- Coupons
- Loyalty
- Rewards

---

## Discount

Applied automatically.

Examples

- 10% Off
- Lunch Menu
- Weekend Offer

---

## Coupon

Applied using a code.

Examples

WELCOME10

FREEGYOZA

SUMMER20

---

## Loyalty

Customers earn points from purchases.

Transactions are stored permanently.

---

## Rewards

Restaurant defines rewards redeemable with loyalty points.
